## Features

- **Responsive Design:** Works on all screen sizes (mobile, tablet, desktop).  
- **Touch Controls:** On-screen D-pad for mobile/tablet gameplay.  
- **Keyboard Controls:** Use arrow keys or WASD on desktops.  
- **Customizable Snake Color:** Pick your favorite color for the snake.  
- **Adjustable Speed:** Increase or decrease the game speed with a slider.  
- **Grid Option:** Toggle a visual grid on or off.  
- **Game Modes:**
  - **Classic:** Standard Snake with collision game over.  
  - **Wraparound:** Snake can pass through edges and appear on the opposite side.  
- **Score Tracking:** Current score displayed on-screen.  
- **Pause/Resume:** Pause the game at any time with a button.  

## How to Play

1. Move the snake using **arrow keys** or the **touch D-pad**.  
2. Eat the red food squares to grow longer.  
3. Avoid colliding with yourself in Classic mode.  
4. Adjust speed, snake color, and grid in the settings sidebar.
